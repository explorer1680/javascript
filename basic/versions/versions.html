Lecture: ES5, ES6

1996: Changed from LiveScript to JavaScript to attract Java developers. JavaScript has almost nothing to do with Java. It is a just are marking reason.

1997: ECMAScript 1 became the first version of the JavaScript Language standard:

  • ECMAScript: The Language standard;
  • JavaScript: The Language in practice

2009: ECMAScript 5 (ES5) was released with lots of new features. But, it took many years for browser to take these changes, and it is only today, we can use ES5 without any restrictions.

2015: ECMAScript 2015 (ES2015, ES6) was released: the biggest update ever

2016: ECMAScript 2016 (ES2016) was released with minor chages only.

ES5 is fully supported in all modern browsers, ready to be used today(2016)

ES6(ES2015), Only partial supported in modern browser, no support in older browsers;
